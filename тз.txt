Исходные данные:  
H — высота лестницы, м  
S — площадь, м2
W — ширина лестницы, м  
Тип лестницы: прямой марш, два прямых маршей, угловая, винтовая, поворотная

Общие расчёты для всех типов лестниц:
 
1. Определение рекомендуемой высоты ступеней (h):  
Рекомендуемая высота ступени 0.17 м или же 170 мм.

2. Определение количества ступеней (n):  
n = (H / h)
h - высота ступени 
H - высота лестницы 

3. Определение шага ступени (t):  
 Горизонтальную длину лестницы L можно определить:  
L = S / W
S - площадь, м2
W - ширина лестницы, м  

4. Расчет шага t ступени :  
t = L / n
L - горизонтальная длина лестницы 
n - количество ступеней 

5. Проверка эргономичного соотношения:  
2h + t ≈ 0.63 м (допустимые отклонения +- 0.03 м)
h - высота ступени 
t - шаг ступени 

Для различных типов лестниц:

1. Лестница с прямым маршем:
Общие формулы и расчёты.

2. Лестница с двумя прямыми маршами:  
   • Разделить количество ступеней на два:  
     n1 = n2 = n / 2  
   • Высота каждого марша:  
     H1 = H / 2

3. Угловая лестница:  
   • Необходимо учитывать уменьшение длины из-за угла.  
   • Угол всегда будет = 90° 
     L = (S / W) - (ширина поворота)

4. Винтовая лестница:  
   • Для винтовых лестниц используется радиус R:
 L = 2 п R (длина окружности) 
п - 3.14 
То есть если лестница винтовая нужно будет еще указать R - длину окружности  
  
5. Поворотная лестница:  
   - Учитывается угол поворота для расчета:  
     L = S / (W + угол°) 
 То есть если поворотная нужно указать угол поворота, либо дать адекватный выбор (90° - Г образна; 180° - П образная; 270° - Полувинтовая)  

h стандарт = 0.17 м
H — высота лестницы (ввод)
S — площадь помещения (ввод)
W — ширина лестницы (ввод)
Тип лестницы (выбор)
